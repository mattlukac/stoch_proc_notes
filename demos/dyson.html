Dyson's Brownian Motion

We need to simulate an \(n \times n\) Hermitian matrix whose upper triangular entries are independent Brownian Motions, (with diagonal having twice the variance as the offdiagonal) and watch how the eigenvalues move.

Here's simulation of one matrix:

efn <- function (B) { eigen(B, symmetric=TRUE, only.values=TRUE)$values }

Binit <- function (n) {
    B0 <- matrix(rnorm(n^2)/sqrt(2), nrow=n) + 1i * matrix(rnorm(n^2)/sqrt(2), nrow=n)
    return( (B0 + Conj(t(B0)))/2 )
}

B0 <- Binit(10)
stopifnot(all(Mod(B0 - t(Conj(B0))) <= 1e-8))
efn(B0)
##  [1]  4.547326261  2.972446789  1.666264818  1.071426023 -0.007814261
##  [6] -1.003161896 -1.227721049 -1.725001215 -3.152735580 -4.186965807

Here's an update function:

dB <- function (B, dt) {
    n <- nrow(B)
    B1 <- sqrt(dt) * (matrix(rnorm(n^2)/sqrt(2), nrow=n) + 1i * matrix(rnorm(n^2)/sqrt(2), nrow=n))
    B1 <- (B1 + Conj(t(B1)))/2
    return(B + B1)
}

B1 <- dB(B0, 1)
stopifnot(all(Mod(B1 - t(Conj(B1))) <= 1e-8))
efn(B1)
##  [1]  5.3996409  3.6135277  2.3364899  0.7737107  0.2211624 -1.4390474
##  [7] -2.3949095 -3.0819791 -4.1011183 -4.5251748

Now let's run this for a few steps:

nsteps <- 100
dt <- 0.1
Blist <- list(B0)
for (k in 2:nsteps) {
    Blist[[k]] <- dB(Blist[[k-1]], dt)
}
Emat <- sapply(Blist, efn)
matplot(t(Emat), type='l', xlab='time', ylab='eigenvalue')

If we rescale by \(\sqrt{t}\) and subtract column means then we can see fluctuations about the trend, maybe. This should be stationary, but it doesn't look like it.

tt <- seq(1, dt*nsteps, length.out=nsteps)
E_rescaled <- sweep(Emat, 2, sqrt(tt), "/")
